####核心关键词#######
#################
- 奶油风
- 治愈感
- 日系手账
- 软萌插画
- 轻拟物设计
- 圆润卡通
- 高级但可爱

视觉要求：

配色：
- 主背景：#F8F5F0
- 奶油白：#FFFDF9
- 蜜桃粉：#F7D6D6
- 牛油果绿：#DDE8CF
- 奶油黄：#F7E8B0
- 雾霾蓝：#D8E7F5
- 焦糖棕：#8B6B52

布局：

- 大量留白
- 卡片式Dashboard
- 圆角 24px
- 柔和阴影
- 无锐利边框
- 温暖舒适

组件设计：

- 卡片像贴纸一样
- 按钮像手账贴纸
- 图表带插画装饰
- 图标采用圆润线条风格
- 导航栏像记事本侧边栏

插画风格：

- 使用奶油小狗IP
- 手绘描边
- 低饱和配色
- 表情治愈
- 出现在卡片和空白区域

动效：

- hover轻微放大
- 漂浮感阴影
- 缓慢柔和过渡
- 不使用科技感动画

禁止：

- 科技蓝UI
- 企业后台风
- Material Design风格
- 深色模式
- 高饱和颜色
- 锐利直角
- 强烈渐变

参考：

- Mofusand
- Pochacco
- Sanrio
- 手账App
- 治愈系记账软件
